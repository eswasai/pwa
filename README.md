# Framespik 


## Overview

Framespik is a Next.js-based web application designed for interactive digital experiences. It includes features such as AI-powered sharing, secure data storage, custom applications, and more.

## **Tech Stack**

- **Frontend**: Next.js (React)
- **Styling**: CSS/SCSS/Bootstrap
- **Backend**: Django
- **Hosting**: (Vercel.)

---

## **Installation & Setup**

### **1. Clone the repository**

```sh
git clone https://github.com/Dev-framespik/webapp.git
cd webapp
```

**Project Structure**
<br/>
**Install dependencies**

```sh

npm install

```

<br/>

**Run the development server**

```sh

npm run dev
```
